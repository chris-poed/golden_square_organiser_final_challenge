### class Diary()
- Keeps a list of diary entries
- Adds a DiaryEntry()
- Returns a list of diary entries
- Finds best entry for reading time

### class DiaryEntry()
- Creates/instantiates a diary entry
- Returns a specified diary entry in a nice format
- count_words
- reading_time

### class TodoList()
- Keeps a list of tasks
- Adds a Todo task to the list
- Returns a list of incomplete tasks
- Returns a list of complete tasks
- Marks all Todos as Complete (give_up)

### class Todo()
- Creates/instantiates a Todo task
- Marks a task as Complete

### class ContactList()
- Keeps a list of contacts
- Searches for phone numbers in Diary()
- Returns a list of phone numbers
